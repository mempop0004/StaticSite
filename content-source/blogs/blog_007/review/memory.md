# Review Memory

- 자동화 글은 수동 흐름과 자동 흐름을 둘 다 적는다.
- 명령어는 바로 복사 가능한 예시를 넣는다.
- 실패했을 때의 복구 방법을 같이 쓴다.
- 기준 글은 유닛 페이지 링크와 마지막 행동 지시를 같이 남긴다.
- 자동화 이미지는 추상 시스템 흐름보다 `로그 묶음`, `상태 카드`, `다음 행동 카드`처럼 직관적인 객체와 배치로 내려야 한다.
- 리뷰 자동화 글은 `메모 더미 -> 규칙 카드 -> 다음 체크리스트` 같은 누적 구조가 직접 보여야 한다.
- 이번 007처럼 001은 대비, 002는 흐름 설명으로 역할이 갈릴 때 적합성이 높다.
