# Review

- 글의 중심을 메타를 “마지막 손질”이 아니라 `제목 / 설명 / 태그`의 서로 다른 역할로 분리하는 데 두면서 production system 글다운 판단 기준이 분명해졌다.
- 2번 섹션이 가장 길고 강해서, 각 필드가 같은 글쓰기 작업이 아니라는 점과 규칙이 없을 때 메타가 왜 계속 흔들리는지가 포스팅다운 밀도로 전개된다.
- `제목은 문제와 보상`, `설명은 클릭 이유`, `태그는 재사용 가능한 묶음`이라는 구분이 남아서 바로 적용 가능한 메타 시스템 기준이 생겼다.
- 이미지 구조는 방향이 맞지만 현재 기준으로는 아쉽다. `001`과 `002` 모두 직접 텍스트가 들어가서, 메타 역할 구분을 글자에 기대고 있다.
- 다음부터 메타 시스템 이미지는 텍스트 없이도 제목 카드, 설명 블록, 태그 묶음의 역할 차이가 보이게 더 구조적으로 내려야 한다.
