# Adsense Readiness Baseline

이 문서는 애드센스 운영을 시작하기 전에 필요한 기본 자산을 정리한다.

## Rule

- 확정: 애드센스 준비물은 블로그 단위가 아니라 사이트 공용 자산부터 갖춘다
- 확정: 운영 단위 블로그보다 먼저 공용 신뢰 자산을 준비한다
- 확정: 공용 자산과 블로그 단위 자산을 구분해서 관리한다

## Shared Required Assets

- `about`
- `privacy policy`
- `contact`
- `ads.txt`
- 기본 사이트 설명과 운영 주체 정보

- 대표 글과 후속 글은 블로그 단위 자산이다

## Done Condition
