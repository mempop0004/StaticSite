# Publish Flow

1. 작성 대상 주제를 정한다
2. 관련 규칙을 `rule/`에서 확인한다
3. 게시글 원고를 `src/content/blog/`에 작성하거나 수정한다
4. 제목, 설명, 날짜, 링크 구조를 점검한다
5. 로컬에서는 가벼운 내용 확인만 한다
6. Codespace에서 `npm install`이 필요한지 확인하고 `npm run build`를 실행한다
7. `checklist/post-publish-checklist.md`를 확인한다
8. 작업 저장소에 Git commit 후 `main`에 push 한다
9. 검증된 변경을 실제 발행 저장소에 반영한다
10. Cloudflare Pages 배포 결과와 실서비스 화면을 확인한다
