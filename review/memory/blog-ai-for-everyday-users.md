# blog-ai-for-everyday-users Review Memory

## Current Upgrade Rules

- 도입부는 원론으로 시작하지 말고 사용자가 실제로 겪는 귀찮은 상황으로 시작한다.
- 각 주요 섹션에는 최소 1개의 짧은 실전 예시를 넣는다.
- 글이 추상적으로 끝나지 않게 바로 적용 가능한 사용 장면을 넣는다.
- 내부 링크는 최소 2개를 목표로 한다.
- title과 description은 `현실적`, `바로 써먹는`, `초보자`, `복잡한 자동화 없이` 같은 실용 혜택 언어를 우선한다.

## Next Review Rule

- 다음 글 리뷰 때는 `구체 예시 수`, `도입부 후킹`, `내부 링크`, `실전성`을 우선 검사한다.
