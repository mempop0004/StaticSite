# Output Rule

이 문서는 블로그 템플레이트의 출력 제한과 검색 제한을 정의한다.

## Language Rule

- 검색은 현재 언어 기준으로만 동작한다
- 영어 페이지는 영어 포스트만 검색한다
- 한국어 페이지는 한국어 포스트만 검색한다

## Post List Rule

- 메인 최신글 첫 화면은 최대 `10개`까지 출력한다
- 추가 출력은 `10개` 단위로 한다
- 총 출력 상한은 `30개`다
- 리스트와 검색 카드의 본문 정렬 기준은 `853px` 본문 폭과 같은 체계를 따른다

## UI Text Rule

- 리스트 제목은 최대 `90자`를 기준으로 한다
- 리스트 설명은 최대 `160자`를 기준으로 한다
- 검색 결과 제목은 최대 `90자`를 기준으로 한다
- 검색 결과 설명은 최대 `160자`를 기준으로 한다

## Search Index Rule

- 검색 인덱스는 `title`, `description`, `tags`, `content excerpt`를 사용한다
- 본문 전체를 무제한으로 넣지 않는다
- 검색용 본문 excerpt는 최대 `600자`까지 저장한다
- 검색 결과는 최대 `30개`까지 출력한다

## Layout Rule

- 본문은 무조건 `1단`으로 출력한다
- 본문 최대 폭은 `853px`로 고정한다
- 표와 그림만 예외적으로 넓은 폭을 허용한다
- 광고는 글을 읽는 흐름을 방해하지 않는 휴식 지점에 배치한다

## Build Rule

- 새 포스트를 추가하면 build 시 검색 인덱스를 자동 생성한다
- 출력 제한과 검색 제한은 템플레이트 기본값으로 본다
