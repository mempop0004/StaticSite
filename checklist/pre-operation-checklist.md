# Pre-Operation Checklist

- 글 작성 작업 파일이 실제로 열린다
- ko/en 생성이 정상 동작한다
- 대표 이미지 또는 placeholder가 정상 연결된다
- `/blog`, `/ko/blog` 전체 목록과 페이지가 정상 동작한다
- `npm run build`가 통과한다
- `npm run review:post -- <slug>`가 리뷰 파일을 생성한다
- 리뷰 결과를 보고 현재 글 수정 사항을 반영했다
- 다음 글에 반영할 규칙을 `review/memory/`에 기록했다
- 새 블로그 주제를 운영 단위로 정의했다
- 블로그별 폴더 구조를 운영 직전 기준으로 확정했다
